The hallucination that hurts

Correct pattern. Correct library. Wrong detail.

A parameter that changed two versions ago. A concurrency fix that looks elegant and quietly reintroduces a race. An infra block that reads like real documentation but doesn't exist. It compiles. It passes your smell test. It ships. You find out at 2am.

Cross-checking in a second terminal is too much friction to do every time. So you only do it for decisions you already think are risky. The ones that burn you are the ones you didn't think were risky.

Where to look twice

You don't need three models to be right. You need them different enough to catch each other.

Agreement is useful triage — a strong signal you're on the right track. Disagreement is the real value. It surfaces tradeoffs you'd have missed, edge cases one sees and another doesn't, or a hallucination the others don't share.

The short version

Single Go binary. Stdio MCP. No daemon.

Your agent fork/execs Roundtable per session over stdin/stdout. It subprocesses Claude and Gemini per call, keeps a warm JSON-RPC link to Codex, and POSTs to OpenAI-compatible providers over HTTP. Atomic kill on timeout; processes die with the parent. Cross-platform: Linux, macOS.

Full architecture detail lives in docs/ARCHITECTURE.md. Tool parameters and dispatch options in SKILL.md.
