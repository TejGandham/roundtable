# References

Place external reference materials here that agents need for context:

- **llms.txt files** — machine-readable documentation for libraries and frameworks
- **Design system references** — exported design tokens, component specs
- **API documentation** — external API specs that agents need to integrate with
- **Mockups / wireframes** — HTML comps from design sessions

These files are read-only context. Agents reference them but don't modify them.
